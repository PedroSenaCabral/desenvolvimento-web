HTML

  • É uma Linguagem de Programação Marcação de Hipertexto, do inglês HyperText Markup Language.

  • Possui uma estrutura básica com as tags: doctype, html, head e body.

  • É escrita e formatada com uso de Tags que possuem abertura e fechamento, com exceção das tags órfãs que só possuem abertura.

  • O código HTML ignora espaços em branco, para ter espaços na página também são usadas tags.

  • Listas não ordenadas possuem pontos como marcação e listas ordenadas, números. No código, temos as tags ul para não ordenadas e ol para ordenadas.
