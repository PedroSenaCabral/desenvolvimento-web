CSS

  • CSS, Cascadind Style Sheets, é importante para formatar visualmente o site.

  • É recomendado utilizar o arquivo CSS de forma extrerna. Permitindo usar em diversas páginas html.
    <link rel="stylesheet" type="text/css" href="link do arquivo css">

Estilos

  • Inline: Coloca a formatação direto na linha da tag, mas isso dificulta a manutenção do código.
    <tag style="atributos">

  • Seletores: Coloca a tag style no head aplicando a formatação a todos os componenentes de uma vez.
    Dessa forma: tag { atributos }

  • Classes: Formatar as tags que incluirem essa classe, ela pode ser aplicada inúmeras vezes.
    A classe é escrita assim: .nome da classe {atributos}

  • ID: IDs têm o mesmo objetivo das classes. A diferença é que devem ser usadas para um único elemento da página.
    A ID é escrita assim: #nome do ID {atributos}

  • Span: São elementos inline. Matem o texto formatado na linha.

  • Divs: Utilizada para dividir a página em áreas que podem ser formatadas de forma diferente.São elementos em bloco, e tem precedência em relação a span.

Bordas

    São três valores são passados pelo atributo border. Também é permitido que esses valores sejam passados por atributos individuais, no entanto, usar um pode ser mais simples.

    Exemplo: border: espessura cor tipo-da-borda;

    Outras propriedades sobre as bordas podem ser vistos aqui.

Fontes e Cores

  • É possível alterar a cor dos elementos, com os códigos das cores.

  • Também podemos alterar a fonte dos textos.
    Exemplo dos valores passados no atributo: font-family: {"font-family", generic-family ;}.

    Podemos ver várias propriedades sobre fontes nesses links:
    Família de fontes e propriedades
    Fontes seguras

  • O tamanho das fontes pode ser definido nos seguintes formatos:

    1. px: pixels, o tamanho default é 16px
    2. %: porcentagem, 100% equivale ao tamanho default
    3. em: 1em equivale ao default, e caso é proporcional ao tamanho do elemento definido acima.

  • text-decoration: coloca underline e outros estilos do tipo.

  • Com o atributo font é possível mudar tipo, estilo e tamanho da fonte. Assim como é possível fazê-lo em atributos separados.

Posicionamento

  • Usa-se margin e padding para dar espaçamento entre as divs.

  • Existem elementos dos tipos inline, block e inline-block.

  • Elementos flutuantes: usando a propriedade float os elementos flutuam lado a lado. Para que o texto fique abaixo é preciso usar a propriedade <clear para limpar o fluxo flutuante.

  • Relative: alinha a div em relação a ela mesmo, cujo ponto em consideração é o canto superior esquerdo.

  • Absolute: a div se comporta com se não estivesse dentro da div anterior. É absoluta em relação ao seu parente mais próximo.

  • Fixed: se comporta semelhante ao absolute, no entanto fica relativo com o viewport (área da tela que o usuário enxerga).

Links

  • Não visitados: a:link {atributos}

  • Visitados: a:visited {atributos}

  • Hover: quando passa o cursor sobre o link.
    a:hover {atributos}

  • Ativo: quando clicado.
    a:link {atributos}

  • Para tirar o underline usa o atributo text-decoration.
