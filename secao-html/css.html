CSS

  • CSS, Cascadind Style Sheets, é importante para formatar visualmente o site.

Estilos

  • Inline: Coloca a formatação direto na linha da tag, mas isso dificulta a manutenção do código.
    <tag style="atributos">

  • Seletores: Coloca a tag style no head aplicando a formatação a todos os componenentes de uma vez.
    Dessa forma: tag { atributos }

  • Classes: Formatar as tags que incluirem essa classe, ela pode ser aplicada inúmeras vezes.
    A classe é escrita assim: .nome da classe {atributos}

  • ID: IDs têm o mesmo objetivo das classes. A diferença é que devem ser usadas para um único elemento da página.
    A ID é escrita assim: #nome do ID {atributos}

  • Span: São elementos inline. Matem o texto formatado na linha.

  • Divs: Utilizada para dividir a página em áreas que podem ser formatadas de forma diferente.São elementos em bloco, e tem precedência em relação a span.

Bordas

    São três valores são passados pelo atributo border. Também é permitido que esses valores sejam passados por atributos individuais, no entanto, usar um pode ser mais simples.

    Exemplo: border: espessura cor tipo-da-borda;

    Outras propriedades sobre as bordas podem ser vistos aqui.

Fontes e Cores

  • É possível alterar a cor dos elementos, com os códigos das cores.

  • Também podemos alterar a fonte dos textos.
    Exemplo dos valores passados no atributo: font-family: {"font-family", generic-family ;}.
