HTML

  • É uma Linguagem de Programação Marcação de Hipertexto, do inglês HyperText Markup Language.
  • Possui uma estrutura básica com as tags: <doctype>, <html>, <head> e <body>.
  • É escrita e formatada com uso de tags que possuem <abertura> e </fechamento>, com exceção das tags órfãs que só possuem abertura.
  • O código HTML ignora espaços em branco, para ter espaços na página também são usadas tags.

Imagens

  • É possível inserir imagens com a tag <img> via link para outra página ou tendo a imagem dentro do projeto.

Listas

  • A tag para listas ordenadas é <ol> e para listas não ordenadas, é <ul>.

Tabelas

  • Na criação de tabelas usam-se as tags <table>, <tr> e <td>, onde tr representa as linhas e td as colunas, respectivamente.
  • A largura da tabela pode ser em pixels ou em porcentagem da tela.
  • A propriedade colspan mescla colunas e a rowspan, linhas.
