HTML

  • É uma Linguagem de Programação Marcação de Hipertexto, do inglês HyperText Markup Language.
  • Possui uma estrutura básica com as tags: <doctype>, <html>, <head> e <body>.
  • É escrita e formatada com uso de tags que possuem <abertura> e </fechamento>, com exceção das tags órfãs que só possuem abertura.
  • O código HTML ignora espaços em branco, para ter espaços na página também são usadas tags.

Imagens

  • É possível inserir imagens com a tag <img> via link para outra página ou tendo a imagem dentro do projeto.

Listas

  • A tag para listas ordenadas é <ol> e para listas não ordenadas, é <ul>.

Tabelas

  • Na criação de tabelas usam-se as tags <table>, <tr> e <td>, onde tr representa as linhas e td as colunas, respectivamente.
  • A largura da tabela pode ser em pixels ou em porcentagem da tela.
  • A propriedade colspan mescla colunas e a rowspan, linhas.

HTML 5

    Para o caso de navegadores antigos que podem causar imcompatibilidade tem que colocar o arquivo html5shiv (facilmente encontrado na internet) e um teste condicional no head (ver no inicio desse arquivo).

    No HTML5 existem tags mais específicas para a sua função, têm mais significado. Algumas são:

  • header: cabeçalho.
  • nav: barra de navegação.
  • footer: rodapé.
  • atributo lang no head para atribuir idioma.
  • article: criar postagens.
  • section: criar seções.
  • time: colocar datas e horários.
  • aside: conteúdos laterais à parte principal.

Inputs

    Novas tags inputs html.
  • label: pode associar a label a caixa de texto
  • type email requer um e-mail
  • type number permite apenas números e limitação de valores
  • type url requer uma url
  • placeholder: dica para usuário
  • required: torna o campo obrigatório
  • type search para pesquisa
  • type range botão que arrasta (tipo volume)
  • type data requer uma data
  • type color escolha de cor
